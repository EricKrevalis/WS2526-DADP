You need to:


1. Install Grafana (Linux)

# Ubuntu/Debiansudo apt-get install -y grafanasudo systemctl start grafana-serversudo systemctl enable grafana-server
Or with Docker:
cd grafanadocker compose up -d

2. Open Grafana

Go to http://localhost:3000 (login: admin/admin)

3. Add PostgreSQL Data Source

Connections → Data sources → Add data source
Select PostgreSQL
Settings:
Host: localhost:5432
Database: traffic_lake
User: postgres
Password: (their PostgreSQL password)
SSL Mode: disable
Click Save & Test

4. Import Your Dashboard

Dashboards → New → Import
Click Upload dashboard JSON file
Select grafana/dashboards/pendler-pilot.json
Select the PostgreSQL data source
Click Import
✅ Done! They'll see your exact dashboard!
