# PePi - PendlerPilot

## Our GitHub repository for the DADP project

### File Directory
* Files to run kafka in AWS EC2: [ec2-kafka](./ec2-kafka)
* Files used to connect and use Grafana: [grafana](./grafana)
* Files to test our initial API functions: [ingest](./ingest)
* Files to automate the API calls in AWS: [lambda_functions](./lambda_functions)
* Markdown Notes, such as CMD commands, guides, todos, etc. [notes](./notes)
* Examples of what our "ingest" files had as an output [output_examples](./output_examples)
* Files that our PySpark used to publish and transform data [pyspark](./pyspark)
